# fast-invoice-generator-

## Simplest way to generate invoice

Use this ready-to-copy template and replace the values in `<...>`:

```text
INVOICE
Invoice No: <INV-001>
Date: <YYYY-MM-DD>

From:
<Your Name / Company>
<Address>
<Email / Phone>

Bill To:
<Client Name / Company>
<Client Address>

Item                  Qty    Rate    Amount
<Service/Product>     <1>    <100>   <100>

Subtotal: <100>
Tax (%): <0>
Total: <100>

Notes:
<Payment terms or thank-you note>
```
